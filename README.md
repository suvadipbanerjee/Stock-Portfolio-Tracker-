# 📈 Stock Portfolio Tracker

## 🚀 Goal
Build a simple stock portfolio tracker that calculates the total investment value using manually defined stock prices.

---

## 📌 Project Description
This is a beginner-friendly Python project.

The user enters:
- Stock name
- Quantity of shares

The program uses a hardcoded dictionary to store stock prices and calculates the total investment value.

Example stock price dictionary used in the project:

```python
stock_prices = {
    "AAPL": 180,
    "TSLA": 250,
    "GOOGL": 140,
    "MSFT": 320
}

python project1.py
